Política de Privacidade

Última atualização: junho de 2026

1. Quem somos

A Alcance Marketing é uma empresa de marketing estratégico sediada em Manaus, Amazonas, Brasil. Operamos como agência de gestão de tráfego pago e consultoria de marketing digital para negócios locais.

Para fins de integração com plataformas de terceiros, utilizamos o aplicativo Alcance Marketing API (Meta App ID: 722561597580989) para acessar e processar dados de desempenho de campanhas de anúncios.

2. Quais dados coletamos

Por meio de nossa integração com a API do Meta (Facebook/Instagram), coletamos exclusivamente dados de desempenho de campanhas publicitárias, incluindo:

  • Impressões e alcance de anúncios
  • Cliques e taxa de cliques (CTR)
  • Valor investido (spend) por campanha
  • Conversas iniciadas via WhatsApp
  • Identificadores de contas de anúncios (ad account IDs)

Não coletamos dados pessoais de usuários finais, como nome, e-mail, telefone ou qualquer informação de identificação pessoal de consumidores.

3. Como usamos os dados

Os dados coletados são utilizados exclusivamente para:

  • Gerar relatórios de desempenho para nossos clientes
  • Analisar a eficácia das campanhas de tráfego pago
  • Otimizar estratégias de marketing baseadas em dados reais
  • Consolidar métricas em dashboards internos e externos

Os dados nunca são vendidos, compartilhados com terceiros não autorizados ou utilizados para fins além dos descritos acima.

4. Armazenamento e segurança

Os dados são armazenados em servidores seguros localizados no Brasil (Campinas, SP), com acesso restrito apenas aos membros autorizados da equipe Alcance Marketing. Utilizamos conexões criptografadas (HTTPS/TLS) em toda comunicação com APIs externas.

5. Retenção de dados

Os dados de desempenho de campanhas são retidos pelo período necessário para prestação dos serviços contratados. Clientes podem solicitar a exclusão de seus dados a qualquer momento mediante contato direto.

6. Seus direitos

De acordo com a Lei Geral de Proteção de Dados (LGPD — Lei 13.709/2018), você tem direito a:

  • Acesso aos dados que mantemos sobre você
  • Correção de dados incorretos ou desatualizados
  • Exclusão dos seus dados
  • Portabilidade dos dados
  • Revogação do consentimento a qualquer momento
